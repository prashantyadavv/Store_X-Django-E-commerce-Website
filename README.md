# Store_X – Django E-commerce Website

Store_X is a full-featured E-commerce web application developed using the Django framework.
The platform allows users to browse products, manage their cart, and place orders online.

## Features

* User authentication (Login / Register)
* Product listing and product details
* Add to cart functionality
* Order management
* Admin panel for product management
* Responsive UI

## Technologies Used

* Python
* Django Framework
* HTML
* CSS
* JavaScript
* SQLite Database

## Installation

1. Clone the repository

```
git clone https://github.com/yourusername/store_x.git
```

2. Navigate to the project directory

```
cd store_x
```

3. Install dependencies

```
pip install -r requirements.txt
```

4. Run migrations

```
python manage.py migrate
```

5. Start the server

```
python manage.py runserver
```

6. Open in browser

```
http://127.0.0.1:8000/
```
